## 使用
### clone
```bash
$ git clone http://172.31.160.207:8890/bigdata/big-data-platform/ddh-ui
```
### yarn
```bash
$ yarn install
$ yarn serve
```
### or npm
```
$ npm install
$ npm run serve
```
